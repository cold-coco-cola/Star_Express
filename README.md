# STAR_EXPRESS

## 游戏简介
《STAR_EXPRESS》（译名：《星穹铁道》）是一款太空主题的2D轻策略连线模拟游戏。玩家将作为星穹铁道公司的调度员，在宇安匹人新开拓的星系间规划航线，运送乘客穿越星空，编织高效的银河运输网络。

## 灵感来源
游戏玩法借鉴于《模拟地铁》（《Mini Metro》）,背景设定灵感来源于被地铁折磨的日常体验，部分UI等美术资源来源于Otoke Neko。游戏名称来源于章座林的事迹，作为全国第一个崩铁玩家，本团队希望能在太空中实现他的夙愿，我们的口号是：“星穹铁道，好玩到爆！”

## 背景设定
星穹纪年233年，是宇安匹人（英译yvanpi）走向太空、实现星穹探索的第233年。他们从自己的星系走向了深空，而你，作为星穹铁道公司的新员工，将负责在这些新抵达的星系中规划航线，维系星际客运的运转。

## 玩法介绍
### 1.游戏要素介绍
- **站点**: 白色的几何图形，乘客上车与下车的位置
- **乘客**: 灰色的几何图形，乘客的形状与其目标站点的形状相同
- **轨道**: 链接站点的线段，飞船会在轨道上行驶。不同颜色的轨道代表不同线路，游戏中最多同时使用6种不同颜色的线路
- **飞船**: 每艘飞船ipro会在相同颜色的轨道上往返行驶。游戏画面左侧的第一种资源数量代表目前可调用的飞船数量
- **陨石带**: 宇宙中陨石密布的区域，跨越陨石带的轨道需要消耗星隧来建设
- **星隧**: 用来跨越陨石带的星际隧道，游戏左侧的第三种资源数量为目前剩余的星隧资源数量

### 2.操作介绍
<img width="1108" height="693" alt="image" src="https://github.com/user-attachments/assets/a14464b9-b263-437d-825e-cbb89805a7b2" />

- **建设轨道**: 分别左键单击你想要建设轨道的两个站点，会出现选择轨道颜色的弹窗。选择合适的线路，站点之间会自动生成对应的轨道。
  1. 若规划的线路不合理，或是建设此路线所需要的资源不够，将无法生成路线
  2. 若该轨道跨越了陨石带，将自动消耗一个星隧资源
  3. 若该路线上没有正在运行的飞船，将自动在该路线上添加一个飞船，左侧可调用的飞船数量自动减一
- **删除轨道**: 右键双击轨道可以删除两个站点之间的轨道
  1. 若删除行为不合理，将无法删除
  2. 若想删除该颜色的所有轨道，右键双击屏幕右侧对应颜色的星星图标即可
- **添加飞船**: 左键单击屏幕左侧的飞船图标，会出现可添加飞船的路线，点击你想增加飞船的路线，该路线将自动增加一艘飞船
- **飞船升级**: 屏幕左侧第二个图标显示的资源为飞船升级资源。左键单击该图标，再在屏幕上点击待升级的飞船，该飞船将获得一个船舱，船舱随飞船一起运动，运载能力与飞船相同。同一个飞船最多可以升级四次
- **时间流速调节**: 点击游戏右上角时钟按键，将出现四个选项，分别对应不同的时间流速，再次点击时钟按键将隐藏这四个选项
- **修改路线**: 如果想把一个站点加入已经建立的线路当中，可以右键单击该路线，再左键点击站点，可以在不删除站点的情况下将该站点直接加入路线
  1. 若建立新路线所需的星隧资源不足，将无法生成
- **页面缩放**：使用鼠标滚轮可以缩放游戏画面
- **站点拥挤**：若在站点中的乘客超过站点的承载极限，站点将开始摇晃变色，程度随时间增强，保持拥挤一定时间后，游戏失败
